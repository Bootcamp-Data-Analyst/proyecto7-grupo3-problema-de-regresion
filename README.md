# Car Price Prediction — Proyecto de Regresión (Grupo 3)

Proyecto realizado por Andrés, Thami y Daniel. Este repositorio implementa un flujo completo de machine learning para predecir precios de coches usados (Limpieza, EDA, Preprocesamiento, Modelado con XGBoost y una app Streamlit para predicción y análisis).

- Documentación y análisis completo: `docs/README.md`
- Notebooks: `notebooks/`
- App: `app/streamlit_app.py`
- Modelos guardados: `models/`

Estado: Versión final sin flag `is_luxury`. Para detalles y cómo reproducir, consulte `docs/README.md`.
